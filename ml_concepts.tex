Supervised learning is a type of machine learning where the algorithm learns from labeled training data. The process involves:
\begin{itemize}
    \item Input: Labeled training data (features and their corresponding outputs)
    \item Training: The algorithm learns patterns from the labeled data
    \item Model: The trained algorithm that can make predictions
    \item Prediction: Using the model to make predictions on new data
    \item Feedback: Error correction to improve the model's performance
\end{itemize}
